# Roulette API
## Routes
- POST /login
    - WWW-FORM{"login": "String", "password": "String", "croupier": "String"}
    - Ответы:
      - Code: 441:
        - You already login
      - Code: 443 | 442:
        - Incorrect login or password
      - Code: 435:
        - The croupier already exists
      - Code: 302:
        - Login successful
- GET /login
  - Ответы:
    - Code:200
      - text/html
- GET /protected
  - Ответы:
    - Code:200
      - text/html
    - Code:401
      - Unauthorized
- GET /protected/json
  - Ответы:
    - Code:200
      - JSON{"coins": "Int", "is_croupier": "boolean", "login": "String"}
    - Code:401
      - Unauthorized
- GET /logout
  - Code:200
    - html/text
  - Code:401
    - Unauthorized
- GET /bet
  - Code: 200
    - html/text
  - Code: 401
    - Unauthorized
- POST /bet
  - Code: 200
    - bet accepted
  - Code: 401
    - Unauthorized
  - Code: 403
    - You are croupier. You can't do bets
  - Code: 444
    - Incorrect type or amount
  - Code: 445
    - Insufficient number of coins
- GET /bet/all
  - Code: 200
    - html/text
- GET /start
  - Code: 200
    - html/text
  - Code: 401
    - Unauthorized
  - Code: 403
    - You must be croupier
- GET /result
  - Code: 200
    - html/text
